# Data Professional Survey Breakdown
This project analyzes responses from a survey of data professionals, examining various factors such as job roles, industry, compensation, career satisfaction, and demographics. The dataset was sourced from Kaggle, then cleaned and preprocessed in MS Excel. I used Power BI for data visualization to uncover insights on career transitions, job satisfaction, skills, and challenges in the data field.

# Key Details:

- Data Source: Kaggle
- Tools Used: MS Excel (Data Cleaning), Power BI (Data Visualization)
